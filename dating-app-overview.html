Aura — Dating Platform

"Find your wavelength."

A two-application web platform for modern matchmaking. Users discover and connect with potential partners through a premium, mobile-first interface. Administrators manage member profiles and platform activity from a dedicated control panel.

Proof of Concept React + TypeScript Tailwind CSS Supabase Netlify
Applications
Two dedicated interfaces — one shared backend

Aura — User App

/user-app/

The member-facing dating application. Users log in with a simple username and password, browse other members, like or pass on profiles, and chat with their matches — all in a premium dark-mode interface.

  • Username + password login (PoC — no email required)
  • Discover page: swipe-style profile cards with like / pass
  • Matches page: grid of mutual connections
  • In-app messaging per match
  • My Profile: view and edit personal details & photos
  • 10 pre-seeded demo profiles with ready-to-use credentials

Aura — Admin Panel

/admin-app/  🔒

Private management interface for platform administrators. Access is restricted by a password gate. Provides complete oversight of members, matches, and conversations.

  • Password-protected entry gate
  • Dashboard with KPI cards and signup trend chart
  • Member management — view, edit, suspend or delete profiles
  • Matches overview — all mutual connections with dates
  • Messages moderation — read and delete any conversation
Authentication
Simple login for multi-user PoC simulation

How it works

No real authentication system is used. Each profile has a username and a short plaintext password (max 5 characters) stored in the database. On login, the app looks up the matching profile and saves the profile ID to the browser's localStorage. Logging out clears the session. This allows the demonstrator to switch between any of the 10 pre-built personas and experience the platform from multiple perspectives.

Scope
What is — and is not — included in this PoC

In scope

✓Username / password login & logout
✓Profile discovery with like / pass
✓Automatic match on mutual like
✓In-app messaging per match
✓My Profile — view & edit details
✓10 fully seeded demo personas
✓Admin: member management
✓Admin: matches & messages moderation
✓Premium dark-mode UI, mobile-responsive
✓Deployed on Netlify (free tier)

Out of scope (PoC)

✗Real email / OAuth authentication
✗Push or email notifications
✗Real-time chat (WebSocket / subscriptions)
✗Video or voice calling
✗AI-powered matching algorithm
✗Location-based / map filtering
✗Premium subscription or payment
✗Profile verification / identity checks
✗Native mobile application
✗GDPR compliance tooling
